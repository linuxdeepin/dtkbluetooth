# dtkbluetooth

## 项目介绍

dtkbluetooth是对于deepin/UOS系统上的BlueZ提供的DBus接口的封装，方便应用开发者轻松且快速的调用接口进行开发。

## 项目结构

对外暴露的类和方法均在include文件夹下。

## 如何使用项目

如果要使用此项目，可以阅读相关文档再参照examples文件夹下的例子。
